## 冒烟场景

1. 导入一组合法固定格式日志，看到成功解析摘要。
2. 使用关键字搜索目标日志，打开前后 10 行上下文。
3. 保存一个常用查询，关闭后重新打开仍可调用。
4. 导入合法规则集，系统识别请求列表。
5. 按业务流程、时间、结束结果、是否异常筛选请求。
6. 选择一次请求，查看 A 应用处理、A 到 B 传递、B 应用处理时延。
7. 点击慢阶段，查看阶段边界日志和上下文日志。
8. 查看同类请求统计摘要，判断当前请求是否偏慢。
